Comprehensive Algebra Guide

Algebra is the branch of mathematics that uses symbols, typically letters, to represent numbers and quantities in equations and expressions. It generalizes arithmetic operations and explores relationships through variables, enabling problem-solving across a wide range of applications, from basic linear equations to abstract algebraic structures.

1. Introduction to Algebra

Algebra extends arithmetic by introducing variables (e.g., x, y) to represent unknown quantities. It allows for the formulation and solution of equations, modeling real-world scenarios like physics, economics, and engineering.

  • Key Concept: Variables and constants are combined using arithmetic operations to form expressions (e.g., 3x + 5) and equations (e.g., 2x - 7 = 11).
  • Purpose: Solve for unknowns, analyze relationships, and generalize patterns.

2. Basic Algebra: Expressions and Equations

2.1 Algebraic Expressions

An algebraic expression is a combination of variables, constants, and operations.

  • Monomial: 4x² (single term).
  • Binomial: 3x + 7 (two terms).
  • Polynomial: x³ - 2x² + 5x - 1 (multiple terms).

Operations:

  • Addition/Subtraction: Combine like terms. Example: 5x + 3x - 2x = 6x.
  • Multiplication: Distribute coefficients. Example: 2(x + 3) = 2x + 6.
  • Exponentiation: Apply exponent rules. Example: (x²)³ = x⁶.

2.2 Linear Equations

A linear equation has variables with degree 1. General form: ax + b = c.

Solving Linear Equations:

  1. Isolate the variable using inverse operations.
  2. Example: Solve 2x + 3 = 11:
    • Subtract 3: 2x = 8.
    • Divide by 2: x = 4.

Systems of Linear Equations: Two or more equations solved simultaneously. Example:

2x + y = 5
x - y = 1

Methods:

  • Substitution: Solve one equation for a variable and substitute. From x - y = 1, get x = y + 1. Substitute into 2x + y = 5: 2(y + 1) + y = 5 → 3y + 2 = 5 → y = 1. Then x = 1 + 1 = 2.
  • Elimination: Add/subtract equations to eliminate a variable. Add: (2x + y) + (x - y) = 5 + 1 → 3x = 6 → x = 2. Substitute: 2(2) + y = 5 → y = 1.
  • Solution: (x, y) = (2, 1).

2.3 Inequalities

Inequalities compare expressions using <, >, ≤, or ≥. Example: Solve 3x - 4 < 8:

  • Add 4: 3x < 12.
  • Divide by 3: x < 4.
  • Note: Multiplying/dividing by a negative reverses the inequality. Example: -2x > 6 → x < -3.

3. Intermediate Algebra: Polynomials and Factoring

3.1 Polynomials

A polynomial is an expression with terms of the form ax^n, where n is a non-negative integer. Example: 3x^4 - 2x² + x - 5.

  • Degree: Highest exponent (e.g., degree of x³ + 2x is 3).
  • Operations:
    • Addition/Subtraction: Combine like terms. Example: (x² + 3x) + (2x² - x) = 3x² + 2x.
    • Multiplication: Use distributive property. Example: (x + 2)(x + 3) = x² + 5x + 6.
    • Division: Polynomial long division or synthetic division for linear divisors.

3.2 Factoring Polynomials

Factoring expresses a polynomial as a product of simpler polynomials.

  • Greatest Common Factor (GCF): Factor out GCF. Example: 6x² + 9x = 3x(2x + 3).
  • Difference of Squares: a² - b² = (a - b)(a + b). Example: x² - 16 = (x - 4)(x + 4).
  • Quadratic Trinomials: For ax² + bx + c, find factors of a·c that add to b. Example: x² + 5x + 6 = (x + 2)(x + 3).
  • Sum/Difference of Cubes:
    • a³ + b³ = (a + b)(a² - ab + b²).
    • a³ - b³ = (a - b)(a² + ab + b²).

3.3 Quadratic Equations

A quadratic equation has the form ax² + bx + c = 0. Solutions via:

  • Factoring: Example: x² - 5x + 6 = 0 → (x - 2)(x - 3) = 0 → x = 2, 3.
  • Quadratic Formula: x = [-b ± √(b² - 4ac)]/(2a). Example: For 2x² + 4x - 6 = 0:
    • a = 2, b = 4, c = -6.
    • Discriminant: b² - 4ac = 4² - 4·2·(-6) = 16 + 48 = 64.
    • x = [-4 ± √64]/(2·2) = [-4 ± 8]/4 → x = 1 or x = -3.
  • Completing the Square: Example: x² + 6x - 7 = 0:
    • Move constant: x² + 6x = 7.
    • Add (6/2)² = 9: x² + 6x + 9 = 16.
    • Factor: (x + 3)² = 16 → x + 3 = ±4 → x = 1, -7.

4. Advanced Algebra: Functions and Graphs

4.1 Functions

A function f(x) maps each input x to a unique output. Notation: y = f(x).

  • Linear Functions: f(x) = mx + b, where m is slope, b is y-intercept. Example: f(x) = 2x + 3.
  • Quadratic Functions: f(x) = ax² + bx + c. Graph is a parabola. Vertex at x = -b/(2a).
  • Polynomial Functions: f(x) = a_n x^n + ... + a_1 x + a_0. Degree determines behavior.
  • Rational Functions: f(x) = p(x)/q(x). Asymptotes where q(x) = 0.

4.2 Graphing

  • Linear: Straight line. Example: y = 2x + 1 (slope 2, y-intercept 1).
  • Quadratic: Parabola. Example: y = x² - 4x + 3 has vertex at x = -(-4)/(2·1) = 2, y = 2² - 4·2 + 3 = -1.
  • Intercepts:
    • x-intercept: Set y = 0 and solve.
    • y-intercept: Set x = 0 and evaluate.
  • Asymptotes: For rational functions, vertical at q(x) = 0, horizontal based on degrees of p(x) and q(x).

4.3 Transformations

Transform functions by shifting, stretching, or reflecting:

  • Vertical Shift: f(x) + c shifts up by c (down if negative).
  • Horizontal Shift: f(x - c) shifts right by c (left if negative).
  • Stretch/Compression: a·f(x) stretches vertically by |a| (compresses if |a| < 1).
  • Reflection: -f(x) reflects over x-axis, f(-x) over y-axis.

5. Complex Algebra: Matrices and Determinants

5.1 Matrices

A matrix is a rectangular array of numbers. Example:

[ 1 2 ]
[ 3 4 ]

Operations:

  • Addition/Subtraction: Element-wise for same-sized matrices.
  • Scalar Multiplication: Multiply each element by a scalar.
  • Matrix Multiplication: For matrices A (m×n) and B (n×p), product is m×p. Example:

    • [ 1 2 ] × [ 5 6 ] = [ 1·5 + 2·7 1·6 + 2·8 ] = [ 19 22 ]
    [ 3 4 ] [ 7 8 ] [ 3·5 + 4·7 3·6 + 4·8 ] [ 43 50 ]

  • Applications: Solve systems of equations, transformations, data analysis.

5.2 Determinants

For a 2×2 matrix [ a b; c d ], determinant is ad - bc. Example:

  • For [ 1 2; 3 4 ], determinant = 1·4 - 2·3 = 4 - 6 = -2.
  • Uses: Solve systems (Cramer’s Rule), find inverses, analyze transformations.

5.3 Matrix Inverse

For a 2×2 matrix [ a b; c d ] with non-zero determinant, inverse is:

[ d -b ] / (ad - bc)
[ -c a ]

Example: Inverse of [ 1 2; 3 4 ] is [ 4 -2; -3 1 ] / (-2) = [ -2 1; 1.5 -0.5 ].

6. Abstract Algebra: Groups, Rings, and Fields

6.1 Groups

A group is a set with a binary operation satisfying:

  • Closure: Operation keeps elements in the set.
  • Associativity: (a · b) · c = a · (b · c).
  • Identity: Exists e such that a · e = a.
  • Inverses: For each a, there’s a⁻¹ such that a · a⁻¹ = e.
  • Example: Integers under addition (identity 0, inverse -a).

6.2 Rings

A ring has two operations (addition and multiplication) with:

  • Addition forming an abelian group.
  • Multiplication being associative with an identity (1).
  • Example: Integers ℤ with addition and multiplication.

6.3 Fields

A field is a ring where non-zero elements have multiplicative inverses. Example: Real numbers ℝ, complex numbers ℂ.

7. Key Algebraic Equations and Identities

  • Binomial Theorem: (a + b)^n = Σ [n choose k] a^(n-k) b^k, where [n choose k] = n!/(k!(n-k)!). Example: (x + 2)² = x² + 4x + 4.
  • Quadratic Formula: x = [-b ± √(b² - 4ac)]/(2a).
  • Polynomial Remainder Theorem: If polynomial p(x) is divided by x - c, remainder is p(c).
  • Fundamental Theorem of Algebra: Every non-constant polynomial has at least one complex root.
  • AM-GM Inequality: For positive numbers a_1, ..., a_n, (a_1 + ... + a_n)/n ≥ (a_1·...·a_n)^(1/n).

8. Applications of Algebra

  • Physics: Solve equations for motion, energy. Example: s = ut + (1/2)at² (kinematics).
  • Computer Science: Algorithms, cryptography (e.g., RSA uses modular arithmetic).
  • Economics: Linear models for supply/demand, optimization.
  • Engineering: Matrices for structural analysis, control systems.
