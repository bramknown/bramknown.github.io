Fundamentals of Probability, Combinatorics, Game Theory, and Statistics

Probability

Probability quantifies the likelihood of events occurring, expressed as a value between 0 (impossible) and 1 (certain). It is foundational in statistics, decision-making, and risk analysis.

Key Concepts and Equations

  • Sample Space (S): The set of all possible outcomes of an experiment. Example: For a die roll, S = {1, 2, 3, 4, 5, 6}.
  • Event (E): A subset of the sample space. Example: Rolling an even number, E = {2, 4, 6}.
  • Probability of an Event: P(E) = |E| / |S| (for equally likely outcomes). Example: P(even) = 3/6 = 1/2.
  • Complement: P(E') = 1 − P(E). Example: P(not even) = 1 − 1/2 = 1/2.
  • Union of Events: P(A ∪ B) = P(A) + P(B) − P(A ∩ B). For mutually exclusive events, P(A ∪ B) = P(A) + P(B).
  • Intersection (Joint Probability): P(A ∩ B) = P(A) × P(B) for independent events.
  • Conditional Probability: P(A|B) = P(A ∩ B) / P(B), where P(B) ≠ 0. Example: Given a deck of 52 cards, P(ace | heart) = 1/13.
  • Bayes’ Theorem: P(A|B) = [P(B|A) × P(A)] / P(B). Used for updating probabilities based on new information.

P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}

Example: A bag contains 3 red and 2 blue balls. Probability of drawing a red ball: P(red) = 3/5. If drawing two balls without replacement, probability both are red: P(red1 ∩ red2) = (3/5) × (2/4) = 6/20 = 3/10.

Combinatorics

Combinatorics studies counting, arrangements, and combinations of objects, crucial for probability calculations and optimization problems.

Key Concepts and Equations

  • Factorial: n! = n × (n−1) × … × 1. Example: 5! = 5 × 4 × 3 × 2 × 1 = 120. Note: 0! = 1.
  • Permutations (Order Matters): Number of ways to arrange r objects from n: P(n,r) = n! / (n−r)!. Example: Arranging 3 of 5 books: P(5,3) = 5! / 2! = 60.
  • Combinations (Order Doesn’t Matter): Number of ways to choose r objects from n: C(n,r) = n! / [r! × (n−r)!]. Example: Choosing 2 cards from 5: C(5,2) = 5! / (2! × 3!) = 10.
  • Binomial Theorem: Expands (x + y)^n = Σ [C(n,k) × x^(n−k) × y^k] for k = 0 to n. Example: (x + y)² = x² + 2xy + y².

C(n,r) = \frac{n!}{r! \cdot (n-r)!}

Example: In a lottery, choosing 6 numbers from 49: C(49,6) = 49! / (6! × 43!) ≈ 13,983,816 possible combinations.

Game Theory

Game theory analyzes strategic interactions where outcomes depend on players’ decisions. It is used in economics, biology, and decision-making.

Key Concepts and Equations

  • Game: Involves players, strategies, and payoffs. Types include cooperative/non-cooperative, zero-sum/non-zero-sum.
  • Payoff Matrix: Shows outcomes for each player’s strategy combination. Example: Prisoner’s Dilemma.
  • Nash Equilibrium: A state where no player can improve their payoff by unilaterally changing their strategy. Formally, for strategies s_i, no player i benefits by switching to s_i' given others’ strategies.
  • Dominant Strategy: A strategy that is best regardless of others’ choices. Example: In Prisoner’s Dilemma, confessing is dominant.
  • Minimax (Zero-Sum Games): Players minimize their maximum loss. max(min(payoff)) for one player, min(max(loss)) for the opponent.

\text{Nash Equilibrium: } \forall i, u_i(s_i^*, s_{-i}^*) \geq u_i(s_i, s_{-i}^*) \text{ for all } s_i

Example: Prisoner’s Dilemma Payoff Matrix

Defect(5,0)(1,1)

Payoffs (Player 1, Player 2). Nash equilibrium: both defect (1,1).

Statistics

Statistics involves collecting, analyzing, and interpreting data to make informed decisions. It describes data (descriptive statistics) and infers properties about populations (inferential statistics).

Key Concepts and Equations

  • Mean (Average): μ = (Σx_i) / n for a sample of n values. Example: For {2, 3, 5}, μ = (2 + 3 + 5) / 3 = 3.33.
  • Median: Middle value when data is ordered. Example: For {2, 3, 5}, median = 3.
  • Mode: Most frequent value. Example: For {1, 2, 2, 3}, mode = 2.
  • Variance: Measures data spread. σ² = Σ(x_i − μ)² / n (population) or s² = Σ(x_i − x̄)² / (n−1) (sample). Example: For {2, 3, 4}, μ = 3, σ² = [(2−3)² + (3−3)² + (4−3)²] / 3 = 0.67.
  • Standard Deviation: σ = √σ². Example: σ = √0.67 ≈ 0.82.
  • Probability Distributions:
    • Binomial Distribution: P(X = k) = C(n,k) × p^k × (1−p)^(n−k), for n trials, probability p. Example: 3 coin flips, probability of 2 heads: C(3,2) × (0.5)² × (0.5)¹ = 0.375.
    • Normal Distribution: f(x) = (1 / (σ√(2π))) × e^−((x−μ)²/(2σ²)). Bell-shaped curve, defined by mean μ and standard deviation σ.

f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}

Z-Score: Standardizes data: z = (x − μ) / σ. Example: Score of 85, μ = 80, σ = 5, z = (85 − 80) / 5 = 1.
